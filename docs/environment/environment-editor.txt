テキストエディターをインストールする
====================================================================================================
UTF-8 を扱えるテキストエディターであれば何でも良いです。ここでは Visual Studio Code を紹介します。

1. ブラウザーで `Visual Studio Code – コード エディター | Microsoft Azure <https://azure.microsoft.com/ja-jp/products/visual-studio-code/>`_ のページを開く　→　``今すぐ無料でダウンロードする`` をクリック
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
.. image:: img/2020-03-01_08h17_28.png

2. Windows 用の ``64 bit`` をクリック
----------------------------------------------------------------------------------------------------
※ この手順では System Installer を使用しています。

.. image:: img/2020-03-01_08h17_54.png

.. note::

   - インストールするパソコンを自分だけが使用しているのであれば User Installer / System Installer のどちらでも変わりはないです。
   - インストールするパソコンを他の方と共同で使用しているのであれば使用環境（条件）によりますが User Installer が望ましいと思います。
     
   User Installer
      - インストールしたユーザーだけが使用できる
      - Windows にログイン中のユーザーの領域 ( "%HOMEPATH%" フォルダー内 )にインストールする
   System Installer
      - 誰でも使用できる
      - "C:\Program Files" フォルダー内にインストールする
      - インストール時に管理者権限が必要

3. ``ファイルを開く`` をクリック
----------------------------------------------------------------------------------------------------
.. image:: img/2020-03-01_08h18_24.png

4. ``同意する(A)`` を選択　→　:guilabel:`次へ(N)` をクリック
----------------------------------------------------------------------------------------------------
.. image:: img/2020-03-01_08h19_13.png

5. :guilabel:`次へ(N)` をクリック
----------------------------------------------------------------------------------------------------
.. image:: img/2020-03-01_08h19_31.png

6. :guilabel:`次へ(N)` をクリック
----------------------------------------------------------------------------------------------------
.. image:: img/2020-03-01_08h19_45.png

7. 追加タスクは必要なものをチェック　→　:guilabel:`次へ(N)` をクリック
----------------------------------------------------------------------------------------------------
.. image:: img/2020-03-01_08h20_15.png

8. :guilabel:`インストール(I)` をクリック
----------------------------------------------------------------------------------------------------
.. image:: img/2020-03-01_08h20_26.png

9. インストール中
----------------------------------------------------------------------------------------------------
.. image:: img/2020-03-01_08h20_37.png

10. ``Visual Studio Code を実行する`` のチェックを外す　→　:guilabel:`完了(F)` をクリック
----------------------------------------------------------------------------------------------------
.. image:: img/2020-03-01_08h20_53.png

11. インストールが終了するとメニューに "Visual Studio Code" が登録されます
----------------------------------------------------------------------------------------------------
.. image:: img/2020-03-01_08h22_13.png
