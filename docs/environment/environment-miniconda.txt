Miniconda をインストールする
====================================================================================================

1. ブラウザーで `Miniconda <https://docs.conda.io/en/latest/miniconda.html>`_ のページを開く　→　``Miniconda3 Windows 64-bit`` をクリック
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
.. image:: img/2020-02-29_20h35_52.png

2. ``ファイルを開く`` をクリック
----------------------------------------------------------------------------------------------------
.. image:: img/2020-02-29_20h36_24.png

3. :guilabel:`Next` をクリック
----------------------------------------------------------------------------------------------------
.. image:: img/2020-02-29_20h36_42.png

4. :guilabel:`I Agree` をクリック
----------------------------------------------------------------------------------------------------
.. image:: img/2020-02-29_20h36_56.png

5. :guilabel:`Next` をクリック
----------------------------------------------------------------------------------------------------
.. image:: img/2020-02-29_20h37_02.png

6. :guilabel:`Next` をクリック
----------------------------------------------------------------------------------------------------
.. image:: img/2020-02-29_20h37_09.png

7. :guilabel:`Install` をクリック
----------------------------------------------------------------------------------------------------
.. image:: img/2020-02-29_20h37_16.png

8. インストール中
----------------------------------------------------------------------------------------------------
.. image:: img/2020-02-29_20h37_22.png

9. :guilabel:`Next` をクリック
----------------------------------------------------------------------------------------------------
.. image:: img/2020-02-29_20h38_01.png

10. ``Learn more ・・・`` と ``Learn how ・・・`` のチェックを外す　→　:guilabel:`Finish` をクリック
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

.. image:: img/2020-02-29_20h38_14.png

11. Miniconda のインストールが終了するとメニューに "Anaconda Powershell Prompt(Miniconda3)" と "Anaconda Prompt(Miniconda3)" が登録されます
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
.. image:: img/2020-02-29_20h38_43.png
