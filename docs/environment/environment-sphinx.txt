Sphinx をインストールする
====================================================================================================

#. "Anaconda Powershell Prompt(Miniconda3)" を起動

   .. image:: img/2020-02-29_23h00_08.png

   |

#. Sphinx をインストール

   :command:`pip install -U Sphinx`

   .. image:: img/2020-02-29_23h25_49.png

   .. image:: img/2020-02-29_23h27_01.png

   |

#. Sphinx のバージョン確認

   :command:`pip show sphinx`

   .. image:: img/2020-02-29_23h27_20.png
