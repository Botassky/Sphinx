Windows 10 で Sphinx を使う
====================================================================================================

.. _sphinx-about:

Sphinx とは
----------------------------------------------------------------------------------------------------
- Python で作成されたオンラインドキュメント作成ツール
- マニュアルなどの記述に向いている
- reStructuredText で記述したテキストファイルを HTML 形式に変換して公開する
- LaTeX と組み合わせることで PDF 形式に変換も可能

.. _sphinx-enviroment:

検証環境
----------------------------------------------------------------------------------------------------
- Microsoft Windows 10 Pro Version 1909 (64-bit)
- Miniconda
- Sphinx 2.4.3
