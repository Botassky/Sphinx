  • グリッドテーブルを使用すると複雑なテーブルを作成できます。

+-----+-------+-------+--------+
| and | False | False | False  |

|     | True  | False |        |

|     | False | True  |        |

列幅の変更¶

  • CSV テーブルとリストテーブルの列幅のデフォルトは等幅です。

  • widthsオプションを指定し、列幅を割合で指定できます。

.. csv-table::
   :widths: 1, 1, 2

   "A", "B", "A and B"
   "False", "False", "False"
   "True", "False", "False"
   "False", "True", "False"
   "True", "True", "True"
